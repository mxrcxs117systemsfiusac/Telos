# Telos

## Descripción

Proyecto web desarrollado con React, TypeScript y Vite.
Arquitectura modular para organización y gestión personal.

## Tecnologías

* React
* TypeScript
* Vite

## Módulos

* Inicio
* Billetera
* Horario
* Estudio
* Ingeniería
* Inteligencia Artificial
* Plan
* Ajustes

## Versión

### v3.0.0

* Eliminación de archivos de prueba
* Proyecto base en blanco

## Ejecución

```bash
npm install
npm run dev
```
